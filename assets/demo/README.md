Demo cover assets live here.

Recommended format:
- `.webp`
- `1000x1500` (A little bit lower is fine)
- `2:3` portrait ratio

Folder layout:
- `assets/demo/books/book-id.webp`
- `assets/demo/authors/author-id.webp`

After adding files, register them in:
- `src/demo/covers.ts`
- `src/demo/author.ts`
